
# SkyCast

Welcome to "SkyCast", your go-to destination for the most accurate and up-to-date weather forecast information!




## Installation

Install dependencies

```bash
  npm install 
```
Start the project
```bash
  npm start
```
## Used Packages
-  express
-  dotenv
-  body-parser
-  ejs
-  node-fetch
